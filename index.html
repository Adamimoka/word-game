Word Game




How to Play

When the game starts, a prompt will appear. You must type a word that includes the prompt. For example, if the prompt is "cat", you could type "cat", "caterpillar", or "application".

Press Enter to submit your word. If the word is valid, you will earn points based on the length of the word, the timer will reset, and the prompt will change. If the word is invalid, the timer will not reset and you will have to quickly type a valid word.

You have limited time to type each word, sarting from 10 seconds and slowly reducing. If you run out of time, the game will end.
